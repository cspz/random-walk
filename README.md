# Random Walk & Stochastic Simulation

Small project to practice Python, numerical simulation, and data analysis.
The goal is to simulate random walks, analyze their statistical properties, and visualize the results.  


<div align="center">
	<img src="docs/assets/rand-walk-gif.gif" alt="radar-room demo animation" width="100%">
</div>


---

## Goals

- Simulate 1D and 2D random walks  
- Plot trajectories and distributions  
- Compute mean, variance, and histograms  



---

## Tools

- Python  
- NumPy  
- Matplotlib  
- (Optional later) pandas  

---


## Future Extensions

- Multiple trajectories  
- 2D random walk  
- Time-series statistics  
- Simple stock-price style model
